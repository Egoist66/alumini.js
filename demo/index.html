Alumini Components

A lightweight library of modern web components built with TypeScript and Shadow DOM.

Button

A versatile button component with multiple variants and states.

Default Secondary Ghost Danger 
<al-button>Default</al-button>
<al-button variant="secondary">Secondary</al-button>
<al-button variant="ghost">Ghost</al-button>
<al-button variant="danger">Danger</al-button>

Sizes

Small, medium, and large button variants.

Small Medium Large 
<al-button size="sm">Small</al-button>
<al-button size="md">Medium</al-button>
<al-button size="lg">Large</al-button>

States

Disabled and loading states.

Disabled Loading Full Width 
<al-button disabled>Disabled</al-button>
<al-button loading>Loading</al-button>
<al-button full-width>Full Width</al-button>

Properties

Property Type Default Description
variant string primary Button color variant
size string md Button size
disabled boolean false Disables the button
loading boolean false Shows a loading spinner
full-width boolean false Stretches button to full width

Accordion

Expandable content sections with smooth animations.

Alumini is a lightweight web component library built with TypeScript and Shadow DOM for encapsulation.

Simply include the built script in your HTML and use the custom elements like any native HTML tag.

Yes! Use CSS custom properties like --al-primary, --al-radius, --al-border to theme the components. 

<al-accordion>
  <al-accordion-item title="Section 1">
    <!-- content here -->
  </al-accordion-item>
  <al-accordion-item title="Section 2" open>
    <!-- content here -->
  </al-accordion-item>
</al-accordion>

Multiple Open

Allow multiple accordion items to be open simultaneously.

This item starts open.

You can open multiple items at once.

Click to expand and collapse independently. 

<al-accordion multiple>
  <al-accordion-item title="Item 1" open>...</al-accordion-item>
  <al-accordion-item title="Item 2">...</al-accordion-item>
</al-accordion>

Properties

Property Type Default Description
multiple boolean false Allow multiple items open
Item Property Type Default Description
title string "" Header text
open boolean false Starts expanded
